# QolayKZ Admin Panel

Админ панель для управления ресторанами, столами, категориями и продуктами.

## Структура

- `index.html` - Главная страница (dashboard) со статистикой
- `restaurants.html` - Управление ресторанами (CRUD)
- `tables.html` - Управление столами (CRUD + генерация QR)
- `categories.html` - Управление категориями (CRUD)
- `products.html` - Управление продуктами (CRUD)
- `css/admin.css` - Стили для админ панели
- `js/admin-api.js` - API клиент для админских операций
- `js/dashboard.js` - Логика главной страницы
- `js/restaurants.js` - Логика страницы ресторанов
- `js/tables.js` - Логика страницы столов
- `js/categories.js` - Логика страницы категорий
- `js/products.js` - Логика страницы продуктов

## Использование

1. Откройте `http://localhost:3000/admin/index.html` в браузере

2. **Главная страница** - показывает статистику и последние рестораны

3. **Рестораны** - создание, редактирование, удаление ресторанов
   - При создании автоматически создается владелец (или используется существующий по телефону)

4. **Столы** - управление столами ресторанов
   - Создание столов с автоматической генерацией QR токена
   - Просмотр и скачивание QR кодов
   - Ротация QR токенов (обновление QR кода)

5. **Категории** - управление категориями меню
   - Мультиязычные названия (RU/KZ/EN)
   - Порядок сортировки

6. **Продукты** - управление блюдами
   - Мультиязычные названия и описания
   - Цена, станция (HOT/COLD/BAR/NONE)
   - Фильтрация по ресторану и категории

## API Endpoints

Админка использует следующие endpoints:

- `GET /admin/restaurants` - Список ресторанов
- `POST /admin/restaurants` - Создать ресторан
- `GET /admin/restaurants/:id` - Получить ресторан
- `PATCH /admin/restaurants/:id` - Обновить ресторан
- `DELETE /admin/restaurants/:id` - Удалить ресторан

- `GET /admin/tables` - Список столов
- `POST /admin/tables` - Создать стол
- `GET /admin/tables/:id/qr` - Скачать QR код
- `POST /admin/tables/:id/rotate-token` - Ротировать QR токен

- `GET /categories` - Список категорий
- `POST /categories` - Создать категорию
- `PATCH /categories/:id` - Обновить категорию
- `DELETE /categories/:id` - Удалить категорию

- `GET /products` - Список продуктов
- `POST /products` - Создать продукт
- `PATCH /products/:id` - Обновить продукт
- `DELETE /products/:id` - Удалить продукт

## Особенности

- **Фильтрация** - фильтрация столов, категорий и продуктов по ресторану
- **QR коды** - генерация и скачивание QR кодов для столов
- **Мультиязычность** - поддержка RU/KZ/EN для категорий и продуктов
- **Модальные окна** - создание и редактирование через модальные окна
- **Валидация** - проверка обязательных полей

## Станции (station)

Продукты могут быть привязаны к станциям:
- `HOT` - Горячая станция
- `COLD` - Холодная станция
- `BAR` - Бар
- `NONE` - Без станции
